Tags HTML

Listas Não Ordenadas

Utiliza marcadores para cada item adicionado à lista

A tag <ul> é utilizada para iniciar uma nova lista não ordenada.
A tag <li> adiciona um novo item à lista.

O código:
<ul>
<li>Item 01
<li>Item 02
<li>Item 03
</ul>
apresenta o seguinte resultado:

  • Item 01
  • Item 02
  • Item 03

Listas Ordenadas

Utiliza numeração para cada item adicionado à lista

A tag <ol> é utilizada para iniciar uma nova lista ordenada.
A tag <li> adiciona um novo item à lista.

O código:
<ol>
<li>Item 01
<li>Item 02
<li>Item 03
</ol>
apresenta o seguinte resultado:

  1. Item 01
  2. Item 02
  3. Item 03

Links

Para criar um link em HTML utiliza-se a tag<a> com o atributo href para referenciar o endereço a ser acessado com o link

O código:
<a href="http://google.com.br">GOOGLE</a>
cria um link para o site do google

Os links podem ter como referência qualquer arquivo, sendo que o navegador decidirá como será tratado o arquivo.
